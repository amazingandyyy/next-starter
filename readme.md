# next-starter ![travisci][travis]

## Usage
```
$ git clone https://github.com/amazingandyyy/next-starter.git
$ yarn
$ npm run dev
-> open browser localhost:3000
```
## deploy to your own now server
```
now
```

## Testing & linter
```
$ npm run test
```
